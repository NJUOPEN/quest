日程管理(Quest)
==============
  本软件专为拖延症患者设计，可帮助您有计划、高效地完成各项既定日程。

  本软件采用GPL v2许可证授权，版权归原作者所有。

使用方法：
------------
1.从[GITHUB]<https://github.com/NJUOPEN/quest>上下载项目文件；

2.64位Linux可直接执行已编译好的二进制文件：

	./quest

3.其他版本操作系统的用户请先编译，然后执行：

	gcc menu.c quest.c SL.c function.c -o quest
	./quest
